# BH1750 Light Logger 
Log data from a BH1750 light sensor to csv on a Raspberry Pi. Made to record the light levels in my garden to determine best lawn variety. 



- BH1750 Sensor
- Raspberry Pi
- Breadboard
- Battery Power Supply
- Clear tub
